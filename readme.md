# Portal de Emergência — Prefeitura de Aracruz

Página de monitoramento dos portais institucionais da Prefeitura de Aracruz, publicada via GitHub Pages.

**URL:** https://prefeituraaracruz.github.io

## Funcionalidades

- Exibe o status em tempo real dos portais principais (`aracruz.es.gov.br` e `servidor.aracruz.es.gov.br`)
- Atualiza automaticamente a cada 1 minuto
- Links de acesso rápido aos sistemas internos
- Interface mobile-first
